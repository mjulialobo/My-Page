Next Level Weel

During the first week of June 2020, Rocketseat offered a course to develop a complete application from scratch. In the Starter level project, I dedicated 5 days to Ecoleta. To build the application we used HTML, CSS, Javascript, Node.js and SQLite.

Ecoleta is a marketplace for waste collection. Through a search, the user can find a nearby location to take different types of waste. The suggested locations for the user are obtained through a registration carried out by the collection points on the website.

Bootcamp Launchbase

The Rocketseat LaunchBase bootcamp is a program to provide a complete base of fullstack development at a junior level through the Javascript ecosystem, passing through frontend, backend, databases and building interfaces in an objective and compatible manner with the market.

The LaunchBase bootcamp has as its base technologies: HTML, CSS, JavaScript, NodeJS, SQL, PostgreSQL The course is still in progress.
